Interactive multi-view dashboard for exploring CDC BRFSS health indicators with demographic breakdowns.

## Features

- **Multiple Views**: Combines several visualization types in a unified layout
- **Demographic Filters**: Drill down by age, gender, income, and other demographics
- **Coordinated Interaction**: Selections in one view update others
