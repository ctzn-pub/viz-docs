## Overview

An advanced Forest Plot for comparing multiple effect estimates. Essential for meta-analyses or reporting coefficients from multiple models.

## Features

- **Ranked Effects**: Can be sorted by effect size.
- **Precision Indicators**: Marker size can be weighted by the inverse of the variance.
