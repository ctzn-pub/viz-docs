## Overview

Precision-weighted dot plot where marker size reflects statistical precision of the odds ratio estimate.

## Features

- **Precision Weighting**: Larger dots indicate more precise estimates (smaller standard errors)
- **Confidence Intervals**: Horizontal lines show uncertainty range
- **Log Scale**: Symmetric display around the null value of 1.0
