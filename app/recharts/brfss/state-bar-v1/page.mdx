## Overview

The `StateBarChart` is optimized for categorical comparisons across US states. It's particularly effective for health surveillance data like the BRFSS.

## Features

- **Ranked Display**: Automatically sorts states by value for quick comparison.
- **National Baseline**: Dynamically highlights how individual states compare to the national average.
- **Accessible Design**: High contrast and WCAG-compliant color scales.

## Data Schema

The component expects an array of state-level data points.

```typescript
interface StateData {
  state_name: string;
  state_abbr: string;
  overall: number;
}
```
