Visualizes the relationship between two variables using a scatter plot and an OLS (Ordinary Least Squares) regression line.

## Statistical Details

- **OLS Line**: Represents the best-fit line.
- **Correlation Strength**: Provides a visual cue for the strength of the relationship.
